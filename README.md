# Demo Project Summary

Demo is a Spring Boot web application built with Java 17 and Maven. This is a beginner-friendly Spring Boot project designed to demonstrate core concepts and best practices.

## Key Features

- **Framework:** Spring Boot 4.0.5
- **Language:** Java 17
- **Build Tool:** Maven (with Maven Wrapper for easy setup)
- **Database:** MySQL integration with MySQL Connector/J
- **API:** RESTful web services using Spring Boot Web MVC
- **Testing:** Includes Spring Boot Web MVC testing support
- **Containerization:** Docker support for easy deployment

## Technology Stack

- Spring Boot Web MVC for building REST endpoints
- MySQL database connectivity
- Java 17 LTS (Long Term Support)
- Maven for dependency management and build automation

## Project Structure

- Maven wrapper scripts (mvnw / mvnw.cmd) for cross-platform builds
- Dockerfile for containerized deployment
- Standard Spring Boot project layout with src/ directory

## Getting Started

1. Clone the repository
2. Ensure Java 17 is installed
3. Run `./mvnw spring-boot:run` (Unix/Mac) or `mvnw.cmd spring-boot:run` (Windows)
4. Build Docker image with the provided Dockerfile for containerized deployment

## Use Case

This project serves as an excellent starting point for developers learning Spring Boot, REST API development, and modern Java web application architecture.
